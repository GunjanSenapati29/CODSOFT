# ATM Interface

This is **Task 3** of my CodSoft Java Development Internship.

## Project Description

The ATM Interface is a simple Java console-based application that allows users to perform basic banking operations such as checking balance, depositing money, and withdrawing money.

The project uses Object-Oriented Programming concepts by creating separate classes for the ATM machine and the user's bank account.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Validate withdrawal amount
- Prevent withdrawal if balance is insufficient
- Display proper success and error messages
- Menu-driven console interface

## Technologies Used

- Java
- Object-Oriented Programming
- Scanner class for user input

## Concepts Used

- Classes and Objects
- Encapsulation
- Methods
- Constructor
- Conditional Statements
- Loops
- User Input Validation

## How It Works

1. The user starts with an initial bank balance.
2. The ATM menu displays options:
   - Check Balance
   - Deposit Amount
   - Withdraw Amount
   - Exit
3. The user selects an option.
4. The ATM performs the selected operation.
5. Proper messages are displayed based on the transaction result.
