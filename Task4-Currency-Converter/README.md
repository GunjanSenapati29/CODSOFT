# Currency Converter

This is Task 4 of my CodSoft Java Development Internship.

## Project Description

The Currency Converter is a Java console-based application that allows users to convert an amount from one currency to another using real-time exchange rates fetched from an online API.

## Features

- Select base currency
- Select target currency
- Enter amount for conversion
- Fetch exchange rates using API
- Convert currency amount
- Display converted amount with currency symbol
- Basic input validation

## Technologies Used

- Java
- REST API
- HTTPURLConnection
- Regex
- Scanner Class
