# 🎓 Student Management System

A Java-based Student Management System developed as part of the **CodSoft Java Development Internship**. This application allows users to manage student records efficiently through a console-based interface.

## 🚀 Features

* Add New Student
* Remove Existing Student
* Search Student by Roll Number
* Edit Student Information
* Display All Students
* File Handling for Data Persistence
* Input Validation
* Menu-Driven Console Interface

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList Collection
* File Handling (File I/O)
* Exception Handling
* Scanner Class

## 📋 Student Information Stored

Each student record contains:

* Name
* Roll Number
* Grade
* Course

## 💻 How to Run

### Compile the Program

```bash
javac StudentManagementSystem.java
```

### Run the Program

```bash
java StudentManagementSystem
```

## 💾 Data Storage

Student records are automatically saved in:

```text
students.txt
```

Example:

```text
Gunjan Senapati,101,A,Computer Science
Rahul Kumar,102,B+,Information Technology
```

The records are loaded automatically whenever the application starts.
